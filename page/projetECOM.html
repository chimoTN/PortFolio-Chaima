Site E-commerce
HTML - CSS - JS - PHP - SQL
Janvier - Septembre 2021


Gread bottle est un site e-commerce dynamique ayant comme produit phare des bouteilles d'air. Le site a été réalisé sur plusieurs mois en plusieurs phases.

On retrouve les pages de base nécessaires au fonctionnement d'un site e-commerce. Celui-ci dispose d'un panier en objet, d'un back-office back-end et front-end.
Photothèque
Détail



Le projet s'est composé en 3 parties avec plusieurs phases. La première se fait en binôme, elle consiste à réaliser le cahier des charges, la création du site, le compte-rendu final. Cette partie étant en binôme, les tâches ont dû être partagées. Un Trello a été mis en place afin de garantir une bonne gestion des tâches et une avancée. Les parties qui suivent se sont faites en autonomie.



La deuxième partie a été de ramener le site statique en dynamique avec un nombre de fonctions, une base de données, l'inscription, la connexion, l'affichage dynamique des produits, un panier fonctionnel, un pdf, un header en dynamique, décomposer le code pour éviter la redondance. Pour finir, nous avons instancié une troisième partie où l'évolution du site s'est faite avec un panier en variable de session, un back-office.
