RASIO
JAVA - SQL - XML - JSON - CSV


Rasio et le premier projet en java ou on doit utiliser une base de donnée. L'application va permettre une gestion des course automobile avec plusieurs fenêtres permettant de manipuler les courses ainsi que les spectateurs , ecuries , employés, tout ça en stockant les informations dans une base de donnée.
Photothèque
Détails



Le projet s'est déroulé sur 2 mois en totale autonomie . Dans un premier temps, la gestion des courses a été demandé avec la création de course , la suppression , la recherche et l'affichage dans un tableau. Suite a cela a été ajouté les écuries avec les mêmes fonctions que course. Les écuries participent à une ou plusieurs courses ainsi, dans course nous pouvons ajouter les écuries participantes. Puis, ce fut le tour de Spectateur et Employé avec le meme procédé. Il y a des spectateurs à une course . Il y a des Employés dans une écurie.



Le grand changement de ce projet fut l'intégration d'une base de données ainsi que de fichier XML, JSON et CSV, tout ca avec la méthode MVC . A chaque création de course , écurie etc nous l'ajoutons dans la base de données, idem pour la récupération de données tous cela via le modèle.

Les fichiers nous ont permis d'avoir un affichage différent des données .
